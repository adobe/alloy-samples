# Message Inbox Demo

This is a demo project for a message inbox using AJO.

## Getting Started

First, install the dependencies.

```bash
npm i
# or
yarn i
# or
pnpm i
# or
bun i
```

Then, start the server

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
