- The documentation for the ChatGPT Apps SDK can be found in the [docs.txt](./docs.txt), the full text is in [./docs-full.txt](./docs-full.txt) and [online](https://developers.openai.com/apps-sdk).
- The backend of this application is a JavaScript + [Hono](https://hono.dev/llms.txt) API, using [`@modelcontextprotocol/sdk` for the MCP server](https://github.com/modelcontextprotocol/typescript-sdk), [zod v4](https://zod.dev/llms.txt) for validation, and React for the frontend.
- The frontend is integrated with the [Adobe Experience Platform Web SDK (`alloy`)](https://github.com/adobe/alloy) ([user documentation](https://experienceleague.adobe.com/en/docs/experience-platform/web-sdk/home#)) and the backend is integrated with the `/v2` Konductor APIs (documented in the Swagger file [`./konductor-api.yaml`](./konductor-api.yaml)).
