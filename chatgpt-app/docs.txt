# ChatGPT Apps SDK (as of 2025-11-03) - https://developers.openai.com/apps-sdk/

> A framework to build integrated applications for ChatGPT using the Model Context Protocol (MCP) Server. The SDK enables developers to create apps with custom tools, conversational flows, and native-feeling user experiences.

Important notes:

- The Apps SDK is currently in preview for building and testing
- The core architecture centers on the Model Context Protocol (MCP) Server
- Apps should be designed to feel native to ChatGPT with quality conversational flows
- Developers must follow quality, safety, and policy guidelines
